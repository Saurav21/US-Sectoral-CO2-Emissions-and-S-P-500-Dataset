This dataset combines daily US sectoral CO₂ emissions data with historical S&P 500 market data to investigate whether real-economy activity indicators contain predictive information about financial markets.
The project focuses on understanding how emissions across sectors such as aviation, transportation, industry, and power generation relate to equity market performance.
This dataset is suitable for:
•	Time-series forecasting
•	Financial machine learning
•	Macroeconomic research
•	Econometric analysis
•	Alternative data research

Content
The dataset includes daily CO₂ emissions measurements across multiple economic sectors:
•	Domestic Aviation
•	International Aviation
•	Ground Transport
•	Industry
•	Power
•	Residential
Historical S&P 500 data was collected using the yfinance Python library and merged using daily timestamps.

Columns
	
date	Observation date
Domestic_Aviation	CO₂ emissions from domestic aviation
International_Aviation	CO₂ emissions from international aviation
Ground_Transport	CO₂ emissions from road transportation
Industry	Industrial sector CO₂ emissions
Power	Power generation CO₂ emissions
Residential	Residential sector CO₂ emissions
SP500	S&P 500 closing index value

Inspiration
Can sectoral emissions act as real-time indicators of economic activity and investor sentiment?
This dataset was created to explore the relationship between macroeconomic activity proxies and stock market dynamics.
